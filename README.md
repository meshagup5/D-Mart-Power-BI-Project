 Overview

Developed an interactive Power BI dashboard to analyze D-Mart’s sales and profitability trends.

Provides key performance indicators (KPIs) for revenue, profit, and orders across various time periods.

Includes filters for Product, Category, Subcategory, and City for easy data exploration.

Features visual insights such as trend charts, category-wise performance, and store profitability.

Enables data-driven decision-making by highlighting the most and least profitable stores.

Pages Included

KPIs View – Displays daily, monthly, quarterly, and yearly performance.

Trends View – Shows detailed profit, revenue, and sales visualizations.
